# Crystal Clear Static Recomp

Static Recompilation of [Shockslayer's](https://github.com/ShockSlayer) Crystal Clear utilizing [Arcanite24's](https://github.com/arcanite24) [GB-Recompiled](https://github.com/arcanite24/gb-recompiled).

---

## NOTE

The source files included are intended to compile to a playable PC executible or Android APK file per the original utility. I included a shell script that compiles the C code byte for byte to Crystal_Clear.gbc. I can't confirm whether or not the code does or doesn't have bugs or remains stable when playing for a long time with all features intact. 

My hope is that the community can utilize what's here to build on given the closed source nature of the original hack.
